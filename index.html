# My CV Website A personal CV website built with plain HTML, CSS, and JavaScript. No frameworks needed. ## How to edit your content Open `index.html` in any text editor. Every piece of text you should change is marked with a comment like this: ```html ``` Search for `EDIT:` to find everything you need to change: - Your name - Your job title - Your bio - Work experience entries - Education entries - Skill names and percentages - Email address and LinkedIn URL - Footer year ## How to add your photo 1. Take a professional photo (square crop works best) 2. Rename it to `photo.jpg` 3. Put it inside the `assets/` folder (create the folder if it doesn't exist) ## How to add your CV as a downloadable PDF 1. Export your CV as a PDF 2. Rename it to `cv.pdf` 3. Put it inside the `assets/` folder ## How to publish it live on GitHub Pages (free) 1. Push all these files to your GitHub repository (main branch) 2. Go to your repository on GitHub 3. Click **Settings** (top menu) 4. In the left sidebar, click **Pages** 5. Under "Source", select **Deploy from a branch** 6. Choose **main** branch and **/ (root)** folder 7. Click **Save** 8. Wait ~1 minute, then your site will be live at: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/` ## File structure ``` index.html ← the main page style.css ← all the styling (colours, fonts, layout) script.js ← scroll effects, animations assets/ photo.jpg ← your profile photo (you add this) cv.pdf ← your CV as PDF (you add this) README.md ← this file ```
